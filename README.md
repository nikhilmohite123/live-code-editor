🚀 Live Code Editor










A powerful and interactive Live Code Editor built using React + Vite that allows users to write and preview HTML, CSS, and JavaScript in real time.

🌐 Live Demo:
👉 https://live-code-editor-sigma.vercel.app/

🚀 Getting Started
1️⃣ Clone the Repository
git clone https://github.com/nikhilmohite123/live-code-editor.git
cd live-code-editor
2️⃣ Install Dependencies
npm install
3️⃣ Run Development Server
npm run dev

App will run at:

http://localhost:5173
🌍 Production Deployment

This project is deployed on Vercel:

👉 https://live-code-editor-sigma.vercel.app/

To deploy your own version:

npm run build

Then:

Upload the dist/ folder to Vercel
OR

Connect your GitHub repository directly to Vercel for automatic deployment

💡 How It Works

Users write code in separate HTML, CSS, and JS panels.

Code is combined dynamically.

Output is rendered instantly inside an iframe.

No page refresh required.
